========================================================================
    应用程序：小型检验科系统
========================================================================
系统初始化设置：

1.选择一台性能较好的电脑作为服务器，在该电脑上安装SQL SERVER 2008或2008以上版本。
========================================================================================================================================
2.打开SQL SERVER，复制安装包中的set.sql并执行，即可在服务器上设置好数据库。
========================================================================================================================================
3.在该电脑上设置好SQL SERVER服务器的访问权限以便其他电脑可以在局域网中连接上该服务器。
========================================================================================================================================
4.在其他电脑上安装小型检验科系统，在数据库设置步骤中填写对应服务器的ip，账户，密码，和数据库名称（默认为LIS）
========================================================================================================================================


管理员模式的默认账户和密码为：
账户：Admin
密码：Admin

当用户在管理员模式添加管理员之后，就不可再使用该账户，而是使用用户自定义的管理员账户
